# ProjectW(Witch)   
장르: 액션, 보스러시
개발 도구: C++ (UE4)
대상 플랫폼: PC(Windows)

Witch Hunt는 한 때 세계를 멸망시키려 했던 마녀들과 싸우는 2.5D 보스러시 게임입니다.
전문적인 마녀사냥꾼이 되어 도망친 마녀들을 사냥하세요.
